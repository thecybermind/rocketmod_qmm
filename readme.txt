1. Install QMM ( https://github.com/thecybermind/qmm2/wiki/Installation )
2. Make a qmmaddons/rocketmod/dlls directory inside your mod directory and place rocketmod_qmm.dll or rocketmod_qmm.so here
3. Add the path to rocketmod_qmm.dll or rocketmod_qmm.so as an entry in the plugins list in qmm2.json
